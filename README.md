# SpaceAPI-workshop2-suvnets-finest1312


SpaceAPI:

GET /api/space/satellite/{city}.jpg. Svarar med bilder på tre förvalda städer (Borås, Göteborg, Stockholm).
GET /api/space/satellite/all. Visar de tre städerna vi valt.
GET /api/space/funfact/{planet}. Returnerar fun fact om angiven planet.
GET /api/space/solarsystem/{planet}. Returnerar information om angiven planet. 
GET /api/space/weather. Returnerar "Fint väder i rymden!"
